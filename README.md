# 🌐 Page AI Translator

**悬浮按钮一键翻译网页，支持 AI 大模型和免费微软翻译，开箱即用。**

装上即用：页面右下角出现悬浮按钮，**单击翻译，长按设置，拖动调位置**。VIA、Kiwi、Edge、Chrome 都能跑。

---

## ✨ 功能

- 🖱️ **悬浮按钮** — 单击翻译，长按设置，拖动调位置
- 🤖 **AI 翻译** — DeepSeek、OpenAI、硅基流动，兼容 OpenAI 格式的都能接
- 🆓 **微软翻译** — 不要钱、不注册、不填密钥，装上就能用
- 📝 **两种模式** — 覆盖原文 / 双语对照（原文下面显示译文）
- 📦 **整句翻译** — 按段落合并文字，不会把一句话拆成几个词瞎翻
- 💾 **自动缓存** — 翻过的句子不再重复请求，省 Token
- 📱 **小屏友好** — 手表手机都能正常滚动操作
- 🎨 **样式隔离** — Shadow DOM，不会跟网页自身的样式打架

## 📥 安装

### 油猴脚本管理器（推荐）

1. 装一个脚本管理器：[Tampermonkey](https://www.tampermonkey.net/) / [Violentmonkey](https://violentmonkey.github.io/) / [ScriptCat](https://docs.scriptcat.org/)
2. 导入 `via_translator.user.js` 即可

### VIA 浏览器

1. 设置 → 脚本 → 右上角 `+` → 导入脚本
2. 安装并启用

### Kiwi / Edge / Chrome 等

1. 先装 Tampermonkey 扩展
2. 再导入脚本

## ⚙️ 设置

### 翻译引擎

| 引擎 | 说明 | 要配啥 |
|------|------|--------|
| 微软翻译 | 免费，Edge 内置接口，即装即用 | 啥都不用 |
| AI 翻译 | 大模型翻译，质量更好 | 填个 API Key |

### AI 翻译怎么配

1. 引擎选「AI 翻译」
2. 填 **API 密钥**（去 [DeepSeek 开放平台](https://platform.deepseek.com/) 申请）
3. 填 **API 地址**（默认 `https://api.deepseek.com/v1`）
4. 点「获取列表」拉模型，挑一个
5. 保存

> 兼容 OpenAI 格式的都能接，比如：
> - `https://api.openai.com/v1`
> - `https://api.siliconflow.cn/v1`
> - 自己搭的兼容接口

### 翻译模式

| 模式 | 长啥样 |
|------|--------|
| 覆盖模式 | 原文直接变成中文 |
| 双语模式 | 原文留着，下面多一行紫色译文 |

## ❓ 常见问题

**Q: 翻译太慢？**
A: AI 翻译要等远程接口，网络不好就慢。换成「微软翻译」会快很多。

**Q: 微软翻译要钱吗？**
A: 不要。走的是 Edge 浏览器的公开翻译接口，跟 Edge 自带翻译是同一套。

**Q: 翻译完了页面乱了？**
A: 脚本用了 Shadow DOM 隔离，不会动网页原本的结构。真出问题刷新一下就行。

**Q: 密钥存哪了？安不安全？**
A: 优先用油猴的 `GM_setValue` 存（沙盒隔离），不支持才回退到 `localStorage`。

**Q: 手表上能用吗？**
A: 能。设置面板全屏可滚动，小屏也填得了东西。

**Q: 翻译结果很奇怪？**
A: 试试切换引擎。AI 翻译质量取决于模型，微软翻译比较稳定。
